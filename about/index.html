关于作者

计算机系网络工程专业。高考报专业时铁了心学计算机，却不知道选哪个专业，只是看网络工程分数高，而且觉得网络黑客很酷；后来发现路由协议和网络端口枯燥乏味，反倒喜欢上了编程。

学校里做过很多不务正业的事：初中到大学都在学生会，依次当过宿管部、学习部、网络部 的部长，管过宿舍风纪，办过全校学术比赛，开发过学校网站；卖过组装PC；打过辩论赛拿过全校冠军。

高中开始接触 VB，之后依次写过 C、Java、ASP、JavaScript、PHP、C++、C#、Android、Python、 go …… 后来认真回想跟计算机最初结缘，可能是小学三年级的 logo 乌龟。毕业后在韩企 S 厂开发过 Android （OS），在互联网公司做过 Java 开发（Server） ，搞过企业内部培训做过 Tech Lead 搭过 pipeline，后来又折腾起 go 。

此博客从大四开始基于 Octopress 更新。

结果韩企安保非常严密，有时还加班比较多，回到家不愿意动键盘，那段时间都没怎么更新……

2016年6月换工作后，建站工具从 Octopress 换成了 Hexo 并重新开张。

现在是一个为奶粉钱发愁的奶爸。

那些年，我写过的博客

我很早就有写博客，要为自己留下一些书面记录。最早的一个博客可以追溯到高中时代。

不过一直没有找到最适合自己的写作方式和辅助工具，博客写写停停，最长时甚至停几年。中间也在一些文学网站呆过一下，最后还是不习惯。

  • 最早因为QQ的便利，写过一个Q-ZONE。后来渐渐觉得过于花俏，逐渐往百度空间转移，现已关闭。
  • 片言·断章：文学博客，主要是整理高中时留下的手稿，多数是诗词。另有一个百度空间作为Q-ZONE的替代，写日常琐事。 由于百度关停百度空间服务，两个都已下线。
  • 中原诗词网 ： 以『田心』为笔名发布过一些文字，高中毕业后再也没上，我一度以为网站可能已经不在了。近期搜索发现还在，只是已经忘记了密码。最近（2020年2月）发现域名还没过期，但网站已经无法访问。

  • 存档Save&Load ：本博客，github-pages，以 软件开发 为主，兼写其它。源码在自己手里，可控性最强。（即使 GitHub 关闭 pages 服务，仍然可以换一个地方托管。）在其他博客陆续下线的时候，以后这里就是主阵地了。

    同时有一个同名的公众号（存档SaveAndLoad，公众号不允许 &）。

  • 智游自在：主题为“玩”，主要是桌游、电子游戏、各种智力玩具等内容。实际上连技术博客尚且断更，这边写了一篇之后就没了下文。由于企业陆续关闭博客服务，这边也不会再投入精力。

  • 信息学院、软件学院辩论队群博：主题是辩论的博客，内容与辩论队队友合写，我暂时是管理员。原本想作为辩论队的理论阵地，资料存档。但是开头之后大家无法保证更新，换届时也没有传承。今年流行微博微信之后，更是无人写博，也就此断更。由于网易关闭博客服务，该博客已下线。

  • 最近留意『简书』。因为支持Markdown，文集的设置，提供一键下载所有文章，允许删除账户 等等良心设定，已注册帐号，后续有可能把一部分内容放在简书作为备份。

在碎片化的时代，先不论内容质量，坚持写字，已属不易。
