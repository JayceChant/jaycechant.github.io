新安装 Linux 上的必要操作

每次新安装完 Linux，都需要进行一些基本的配置和工具安装。可是由于这个操作比较低频，每次回忆一些细节都很费劲，还得重新搜索。

这次重装了某个服务器，顺便记录下来，避免下次重复浪费时间。

确认版本

一般情况下，系统是自己装的，你会知道版本。但也有可能是 VPS 直接提供了一个系统。或者隔太久没登录，忘了。

创建帐号

直接使用 root 进行操作，是相当危险的行为。上来应该赶紧创建一个用户，然后授予 sudo 权限。

基本工具

默认情况下，软件源比较有限，很多工具找不到，需要添加一些额外的源。
